Executive Healthcare Analytics Modernization
Vision

Modern healthcare organizations require trusted, reusable, and well-governed analytics products that support executive decision-making.

This project demonstrates how a traditional SQL-based executive reporting solution can be redesigned using modern analytics engineering principles, enabling reusable business logic, data quality validation, semantic modeling, and executive decision support.

Project Goal

Design a modern executive analytics platform using synthetic healthcare data, dbt, and Lightdash.

The project focuses on architecture, business understanding, and reusable analytics rather than dashboard development alone.

Objectives
Understand executive business questions
Model healthcare operational data
Design reusable business metrics
Build modular dbt transformations
Deliver executive-ready insights
Demonstrate modern analytics engineering practices
